# Realm Source Code

All code used by Unity is located in _RealmClient_

The folders and files within the Unity project are as follows:

```bash
.
└── RealmClient/
    ├── Assets/
    │   ├── _Scripts/ (Central place for code)
    │   │   ├── App/
    │   │   │   ├── Components/
    │   │   │   └── Screens/
    │   │   ├── DataTypes/
    │   │   ├── Maps/
    │   │   ├── Placement/
    │   │   ├── Popup/
    │   │   ├── ScriptableObjects/
    │   │   ├── Static/
    │   │   └── UI/
    │   ├── ARCore_Extensions/ (Configuration for ARCore Extensions library)
    │   ├── AffordanceThemes/ (Theme for shading selected objects)
    │   ├── Data/ (Data for default AR Objects)
    │   ├── Editor/ (Plugins)
    │   ├── ExtensionsAssets/ (More configuration for ARCore Extensions)
    │   ├── Materials/
    │   ├── Models/ (3D models of defualt AR Objects)
    │   ├── Plugins/ (Android setup)
    │   ├── Prefabs/
    │   ├── Resources/
    │   ├── Samples/ (Starter code for ArcGIS)
    │   ├── Scenes/
    │   ├── Settings/ (Configuration for render pipelines)
    │   ├── Shader/
    │   ├── Tests/
    │   ├── TextMesh Pro /(Configuration for TMP)
    │   ├── Textures/
    │   ├── UI/ (Realm Interface UI)
    │   ├── UI Toolkit/ (Configuration for UI Toolkit theme)
    │   ├── UserInterfaces/ (All UI Toolkit interfaces)
    │   ├── XR/
    │   └── XRI/
    ├── Packages/ (Libraries to be installed)
    ├── Project Settings/ (Configuration for project)
    └── UIElementSchema/ (Support for UI Elements)
```
